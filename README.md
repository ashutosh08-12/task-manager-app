#  Task Manager Application
A full-stack Task Management Web Application built using the **MERN Stack (MongoDB, Express.js, React, Node.js)**.  

# Features
- Create new tasks with title and description  
- Edit tasks using a dedicated edit form (modal)  
- Update task status (Pending, In Progress, Completed)  
- Delete tasks **only when marked as Completed**
- RESTful API integration  
- Form validation on both frontend and backend  
- Toast notifications for better user experience  
- Modern responsive UI using Tailwind CSS  
- Clean folder structure  
- Error handling & secure API design 

## Tech Stack
# Frontend
- React (Vite)
- Tailwind CSS
- Axios
- React Toastify
# Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- Express Validator
